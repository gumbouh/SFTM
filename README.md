I have temporarily closed the source code of this project in protest. Our previous LRSL paper (https://aclanthology.org/2024.findings-emnlp.657) was fully open-sourced, but a recent ACL 2026 Main Conference paper (https://arxiv.org/abs/2605.18007) appears to have substantially reused our core ideas, method pipeline, and visual presentation without adequate attribution. Until this issue is properly investigated and resolved, I do not feel comfortable continuing to release code publicly. I support open science, but not the misuse of openness to appropriate others' work.
